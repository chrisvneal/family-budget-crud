# Family Budget CRUD App

A simple one-page React CRUD app for tracking family bills and creditors.  
Built with React + Vite on the frontend and JSON Server as a mock backend.

---

## Features

- Add, view, update, and delete creditors
- Group bills by type (rent, utilities, loans, etc.)
- Store data locally using JSON Server

## Tech Stack

- React (Vite)
